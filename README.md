# News App
A news application. Get the latest news. A mobile application on iOS devices.
Apple devices can run the application.

# Main Features
- Simple news portal
- In app news search
- Bookmark important news
- Filter the news categorywise

# Technologies 
- iOS Platform
- iOS version 16.2
- Xcode 14
- Swift language
- UIKit
- Free News API
